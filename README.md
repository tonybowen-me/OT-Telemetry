# OT-Telemetry

Demonstrations of PILOT — a causal telemetry-integrity layer for OT/ICS.

- **[`pilot-poc/`](pilot-poc/)** — the current demo: PILOT (causal, Layers 1-3) vs
  a Sigma correlational baseline, evaluated on pre-recorded data from a standalone
  third-party [DHALSIM](https://github.com/Critical-Infrastructure-Systems-Lab/DHALSIM)
  water-distribution lab. Deployable free on Render. See
  [`pilot-poc/README.md`](pilot-poc/README.md).
- **`nalco-demo/`** — earlier water-treatment prototype (self-contained simulator).
- **`Reference Docs/`** — source material for the PILOT concept.
